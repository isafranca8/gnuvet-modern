# GNUVet API 🐾

Sistema de gestão veterinária desenvolvido com **FastAPI**, **PostgreSQL** e **SQLAlchemy**.

O objetivo do projeto é fornecer uma API moderna, escalável e segura para gerenciamento de clínicas veterinárias.

Este projeto foi estruturado seguindo princípios de **Clean Architecture**, **Domain Driven Design (DDD)** e boas práticas de engenharia de software.

---

# 🐶 Funcionalidades

O sistema permite gerenciar:

- Clínicas veterinárias
- Usuários do sistema
- Clientes (donos dos animais)
- Pets
- Consultas veterinárias
- Vacinas
- Prontuário médico
- Produtos
- Pagamentos

---

# 📦 Tecnologias utilizadas

Backend:

- Python 3.14
- FastAPI
- SQLAlchemy
- Pydantic v2
- Alembic (migrations)
- PostgreSQL
- Uvicorn

Testes:

- Pytest
- TestClient

Segurança:

- JWT Authentication
- bcrypt password hashing

Infraestrutura:

- Docker
- Docker Compose
